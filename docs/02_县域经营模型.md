>
> 详细数值公式与经济推演见 [06a — 县域经营数值](06a_县域经营数值.md)

## 1. 县域总览

### 1.1 县域结构
```
县城（行政中心）
├── 村庄 ×4~6（按县域类型）
│   └── 各有：人口、耕地、双账本（村民/地主）、地主Agent、村民Agent、有无村塾
├── 集市 ×1~3（按县域类型）
│   └── 各有：名称、商户数、月GMV
├── 县学 ×1（等级制，1→3级）
├── 水利设施（等级制，0→3级）
├── 医疗设施（等级制，0→3级）
├── 衙门（玩家办公地，衙役等级0→3）
└── 附近卫所（军事独立，非县管辖）
```

### 1.2 县域基础属性

| 属性 | 说明 |
|------|------|
| 总人口 | 各村在册村民之和（不含地主在册/隐匿人口） |
| 总耕地（亩） | 各村在册耕地合计（村民耕地 + 地主在册地，不含隐匿地） |
| 县库现金（两） | 可用行政资金 |
| 民心 | 百姓对官府的认可度（0-100） |
| 治安 | 地区安全程度（0-100） |
| 商业 | 商业发展水平（0-100），在GMV计算中作为单商户基础贸易额 |
| 文教 | 教育文化水平（0-100） |
| 地主占地比 | 地主在册土地占总在册土地的比例 |

### 1.3 县域类型（4种）

每局开局由玩家显式选择一种县域类型（测试阶段露出该选项，便于验证不同县域数值演进），决定初始参数和核心矛盾。

| 类型 | 分布地域 | 核心矛盾 |
|------|---------|---------|
| 财赋核心型 | 江南太湖平原 | 高赋税定额与地主占地集中的冲突 |
| 宗族治理型 | 皖南/赣东南山区 | 宗族势力与官府权力的博弈 |
| 沿海治理型 | 闽浙粤沿海 | 海防安全与民生的平衡，财政极度紧张 |
| 黄淮灾荒型 | 黄河淮河中下游 | 灾荒频发与固定赋税的冲突，流民突出 |

**各类型初始参数概览**（基准值，每局 ±20% 随机波动）：

| 参数 | 财赋核心 | 宗族治理 | 沿海 | 黄淮灾荒 |
|------|---------|---------|------|---------|
| 人口 | 8000 | 6000 | 3000 | 5000 |
| 耕地 | 15000亩 | 13000亩 | 5000亩 | 12000亩 |
| 地主占地比 | 63% | 55% | 36% | 47% |
| 县库 | 600两 | 400两 | 150两 | 250两 |
| 上缴比例 | 75% | 65% | 60% | 65% |
| 物价指数 | 1.4 | 1.1 | 0.9 | 0.8 |
| 民心 | 40 | 55 | 45 | 35 |
| 治安 | 60 | 65 | 35 | 40 |
| 商业 | 55 | 30 | 40 | 20 |
| 文教 | 40 | 50 | 20 | 25 |
| 村庄数 | 6 | 6 | 4 | 6 |
| 集市数 | 3 | 2 | 2 | 1 |
| 初始基建 | 学1/水1/医1 | 学1/水1/医1 | 学1/水0/医0 | 学1/水0/医0 |

> 物价指数影响投资成本和部分行政增量，不影响基础行政开支和税收收入。详见 06a §1.4。

## 2. 村庄模型

### 2.1 双账本结构（每村）

每村维护两套独立账本——村民账本和地主账本：

```json
{
  "name": "沈家圩",
  "farmland": 10000,              // 在册耕地总计（村民 + 地主在册）
  "population": 1300,             // 便利字段，等同于 peasant_ledger.registered_population

  "peasant_ledger": {
    "registered_population": 1300, // 在册村民总人数
    "farmland": 3700,              // 村民自耕地
    "grain_surplus": 0.0,          // 村民粮食盈余
    "monthly_consumption": 0.0,    // 村民月消耗
    "monthly_surplus": 0.0         // 村民月均余粮
  },
  "gentry_ledger": {
    "registered_population": 20,   // 地主在册人数
    "hidden_population": 0,        // 地主隐匿人数
    "registered_farmland": 6300,   // 地主在册土地
    "hidden_farmland": 800,        // 地主隐匿土地
    "grain_surplus": 0.0           // 地主粮食盈余
  },

  "gentry_land_pct": 0.63,        // 地主占在册地比例
  "hidden_land_discovered": false, // 该村隐匿土地事件是否已触发
  "land_ceiling": 13200,          // 潜在土地上限 = (farmland+hidden)×1.1~1.4
  "morale": 42,                   // 村级民心
  "security": 58,                 // 村级治安
  "has_school": false              // 有无村塾
}
```

**口径规则**：
- 村承载力上限、县级人均余粮、徭役折银税基——仅使用村民账本
- 农业税税基 = 全部在册耕地（村民自耕地 + 地主在册地）
- 地主账本盈余只记录为地主财富积累，不并入县级人均余粮

### 2.2 过度开发

每村有固定的 `land_ceiling`（建县时生成），当实际耕地（含隐匿地）超过上限90%时，洪灾和旱灾概率递增。详见 06a §2.5。

### 2.3 粮食储备与集市GMV

**粮食储备**（`peasant_grain_reserve`，县级汇总，单位：斤）：
- 初始化 = 年度村民产出 − 4个月消耗（游戏开局为正月）
- 每月扣除 = 村民总人口 × 年消耗(300斤) / 12 × 过度消费乘数
- 五月额外扣除 = 年度徭役折银全额 × 100斤/两（村民无非粮食收入，须卖粮缴纳）
- 九月秋收 += 当年实际村民产出

**集市GMV**（即时计算，无累加）：
```
集市GMV = 商户数 × commercial × 需求系数
需求系数 = clamp(1 + 月均余粮/20, 0.1, 2.0)
```

> 详细公式见 06a §2.3。

## 3. 集市模型

### 3.1 集市属性
```json
{
  "name": "东关集",
  "merchants": 15,     // 商户数（九月年度调整）
  "gmv": 1375.0        // 月贸易总额（即时计算）
}
```

### 3.2 商业税收
```
月度商业税 = 全县月GMV × 商税税率(基准3%，可调1%~5%)
地方留存60%，上缴40%（独立于农业税上缴比例）
```

### 3.3 商户数量年度调整（九月）
- commercial ≥ 60 且 merchants < 30 → merchants += 1
- commercial ≤ 25 且 merchants > 2 → merchants -= 1

## 4. 士绅阶层

### 4.1 定义
拥有科举功名（秀才、举人、进士）或退休官员、官员家属，构成县里的"士绅阶层"。各村地主作为 `GENTRY` 类型 Agent 存在。

### 4.2 地主 Agent 属性结构
```json
{
  "name": "沈伯年",
  "role": "GENTRY",
  "tier": "FULL",
  "attributes": {
    "intelligence": 6, "charisma": 7, "loyalty": 3,
    "personality": {
      "openness": 0.3,
      "conscientiousness": 0.6,
      "agreeableness": 0.2
    },
    "ideology": {
      "reform_vs_tradition": 0.2,
      "people_vs_authority": 0.2,
      "pragmatic_vs_idealist": 0.7
    },
    "reputation": { "scholarly": 50, "political": 70, "popular": 20 },
    "goals": ["扩大家族土地", "维护宗族利益"],
    "bio": "...",
    "backstory": "...",
    "memory": [],
    "player_affinity": 30,
    "village_name": "沈家圩"
  }
}
```

### 4.3 士绅行为模式
- 天然想扩大土地和资源
- 拥有科举功名者享受部分免税特权（增加税收压力）
- 有官场关联的士绅被处理时，可能触发链式事件（上级施压）
- 开明型士绅可能配合公益（修桥、赈灾），但通常附带条件

### 4.4 隐匿土地与户口

**隐匿土地**：
- 开局每村生成 `farmland × 5%~15%` 的隐匿土地，不在册、不纳税
- 水利建设期间若衙役等级≥1，按民心概率触发举报事件（每村仅一次）
- 触发后进入谈判（`HIDDEN_LAND` 类型），与地主协商清丈
- 详见 06a §2.4

**隐匿户口**：
- 地主兼并成功时，按兼并规模比例将在册村民转入 `gentry_ledger.hidden_population`
- 被转移人口从在册户口中消失，不纳税、不服徭役
- V1不提供找回隐匿户口机制

## 5. 军事与治安

### 5.1 卫所制（MVP设定）
- 军队和地方行政完全独立
- 遇山贼、起义等问题，报告附近卫所，军队来负责
- 县令无法直接调动军队，但可以请求支援（需要时间，受知府支持度影响）

### 5.2 县域衙役
- 类似现代辅警，县令可直接管辖
- 通过"增设衙役"投资升级（最高3级），每级治安+8，增加行政开支
- 衙役等级还影响隐匿土地发现概率和兼并阻止能力

## 6. 经济循环

### 6.1 县年度收入

三大税种，分时征收：

| 税种 | 计算方式 | 征收时机 | 上缴规则 |
|------|---------|---------|---------|
| 农业税 | 在册耕地产出 × 税率(9%~15%) × 征收效率 | 九月（秋季） | 按 remit_ratio |
| 徭役折银 | 在册村民 × 0.3两/人·年 | 五月（全额） | 按 remit_ratio |
| 商业税 | 全县月GMV × 商税税率(1%~5%) | 每月 | 固定留存60% |

> 税率可由玩家调整。农业税率调整即时影响民心（每±1%→民心∓3）。
> 商税税率调整影响民心：每+0.5%→民心−1。
> 详细公式见 06a §4.1~§4.3。

**知府年度配额**：每年正月，知府按在册土地和人口下达农业税+徭役折银的上缴定额，
不考虑农业适宜度（按"标准年"估算）。秋季结算时计算配额完成率，影响年度考核。
遭遇灾害时，玩家可申请减免（见 §6.3）。详细公式见 06a §4.5。

**徭役折银与粮食储备**：村民无非粮食收入，须卖粮换银缴纳徭役折银。
每次征收时按 `1两=100斤` 换算，同步从 `peasant_grain_reserve` 扣减。详见 06a §4.2。

### 6.2 县年度支出

| 项目 | 说明 |
|------|------|
| 上缴税额 | 农业税/徭役按 remit_ratio 上缴，商税固定上缴40% |
| 行政开支 | 7项基础开支 + 基建维护，秋季一次性扣除 |
| 投资支出 | 玩家决定，投资时即扣款 |

**行政开支构成**（7项基础 + 基建维护）：

| 项目 | 财赋核心 | 宗族治理 | 沿海 | 灾荒 | 动态变化 |
|------|---------|---------|------|------|---------|
| 官员俸禄 | 50 | 40 | 30 | 35 | 固定 |
| 县丞俸禄 | 25 | 20 | 15 | 18 | 固定 |
| 师爷束脩 | 30 | 25 | 20 | 22 | 固定 |
| 六房书办 | 30 | 20 | 15 | 18 | 固定 |
| 衙役饷银 | 16 | 12 | 8 | 10 | 每次增设 +round(40×PI) |
| 县学经费 | 24 | 18 | 10 | 15 | 每建村塾 +round(10×PI) |
| 衙署杂费 | 25 | 15 | 12 | 12 | 固定 |
| 水利维护 | 按等级 | 按等级 | 0 | 0 | 等级翻倍 |
| 医疗维护 | 按等级 | 按等级 | 0 | 0 | 等级翻倍 |

> PI = price_index。详见 06a §4.5。

### 6.3 灾害减免申请

当年有灾且预计无法完成配额时，玩家可向知府提交一次减免申请，申报一个灾损数额：

- **填大数**：减免多、利民，但难批，严重高报有被查风险（清名受损+知府关系下降）
- **填小数**：容易批，但减免不足，县里实际困难无法充分缓解

详见 06a §4.8。

### 6.4 资金约束
- 县库可为负数
- 当前实现中无"亏空"状态惩罚机制（设计预留）
- 资金不足时无法进行投资

## 7. 投资与建设

> 所有投资成本 = 基准花费 × price_index（基建类还乘以缩放因子）。
> 无AP系统——每月可执行多次投资，仅受资金限制。

### 7.1 投资项目清单

| 项目 | 基准花费 | 见效时间 | 效果 |
|------|---------|---------|------|
| 开垦荒地 | 50两 | 2个月后 | 目标村耕地增加，村民心+5 |
| 修建水利 | 动态(见06a §4.6) | 8/12/18月 | 全县产出+15%/级，灾害减损 |
| 扩建县学 | 动态 | 2/3/5月 | 文教+10/级 |
| 建设医疗 | 动态 | 2/3/5月 | 疫病减轻，人口增长加速 |
| 资助村塾 | 30两 | 4个月 | 目标村文教+5，民心+5 |
| 增设衙役 | 40两 | 即时 | 治安+8，永久增加行政开支 |
| 修缮道路 | 60两 | 2个月 | 商业+max(0, 8−已修次数)，递减 |
| 开设义仓 | 70两 | 即时 | 灾害人口损失×0.65，民心+5 |
| 赈灾救济 | 80两 | 即时 | 灾害人口损失×0.65（与义仓叠加），民心+8 |

### 7.2 基建等级系统

水利、县学、医疗共享统一的等级体系（0→1→2→3）：
- 每升一级，投资费用翻倍：`C × 2^(level-1)`
- 每升一级，年维护费翻倍：`M × 2^(level-1)`
- 建设期间不可继续升级同类
- 详见 06a §1.1b

### 7.3 投资限制
- 开垦荒地：七月八月农忙不可执行；需指定目标村庄
- 资助村塾：需指定目标村庄，已有村塾的不可重复
- 赈灾救济：需当年有灾害，且仅可执行一次
- 基建类：同类在建时不可重复升级
- 衙役/义仓：有上限（衙役3级，义仓1座）

## 8. 核心指标联动

### 8.1 月度指标变化

| 指标 | 变化机制 |
|------|---------|
| 民心 | 基础衰减(−0.33/月) + 文教回升 + 治安联动 − 重税惩罚 |
| 治安 | 基础衰减(−0.33/月) + 衙役加成 + 民心联动 |
| 商业 | 仅受道路修缮(+)和灾害(−)影响，不月度衰减 |
| 文教 | 通过县学等级和村塾提升，不月度衰减 |

> 详细公式见 06a §5.1~§5.3。

### 8.2 关键阈值
- 民心 < 25：触发"民怨沸腾"，暴动风险上升
- 治安 < 20：触发"匪患四起"，治安事件频发
- 商业 > 60：触发商户增长（merchants+1/年）
- 商业 ≤ 25：触发商户流失（merchants−1/年）
- 文教 > 60：概率产出"本县才子中举"事件

## 9. 月度结算流程

游戏以**月**为单位推进（1个月 = 1回合），3年任期共36个月。

```
月度结算流程:
  [正月]  重置年度财政累计 + 知府下达年度上缴配额（按在册土地人口估算，不含灾害因素）
  [二月]  环境漂移（农业适宜度、水患风险、边患风险±小幅随机）
  [每月]  检查并应用已完成的投资
  [每月]  隐匿土地发现检查
  [六月]  灾害判定（洪灾/旱灾/蝗灾/疫病，每年至多一种）
  [每月]  民心月度变化
  [每月]  治安月度变化
  [每月]  地主兼并事件检查
  [五月] 年度徭役征收（全额）
  [九月]  秋季结算 — 人口更新 + 秋收 + 农业税 + 行政开支扣除 + 配额完成率计算
  [每月]  商业更新 — 粮食扣除 → GMV计算 → 月度商税征收
  [腊月]  年终快照 + 清除灾害状态
```

> 详细结算步骤见 06a 附录。

## 10. 全局环境

每县创建时随机生成环境参数（±20%波动），二月漂移：

| 参数 | 说明 | 影响 |
|------|------|------|
| 农业适宜度 | 0.3~1.0 | 直接乘入农业产出 |
| 水患风险 | 0.0~1.0 | 六月洪灾判定基础概率 |
| 边患风险 | 0.0~1.0 | 设计预留（征兵事件） |
