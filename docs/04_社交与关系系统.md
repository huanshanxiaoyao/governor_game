## 1. 好感度体系

### 1.1 基础规则
- 范围：-99 到 99
- 新关系初始值：10
- 关系加成（开局时叠加）：

| 关系类型 | 好感度加成 |
|---------|-----------|
| 同乡（同省） | +10 |
| 同窗（同年进士） | +5 |
| 同部门/上下级 | +10 |
| 师生关系 | +20 |
| 同派系成员 | +15 |
| 对立派系 | -10 |
| 对方政敌的朋友 | -10 |

### 1.2 好感度阈值效果

| 区间 | 关系状态 | 行为表现 |
|------|---------|---------|
| 70+ | 挚友/心腹 | 主动帮忙、分享内部消息、考核力挺、可能形成人情债 |
| 40-69 | 友好 | 正常互助、社交顺畅 |
| 10-39 | 普通 | 公事公办，不主动帮忙也不使绊子 |
| -10~9 | 冷淡 | 不配合，忽略你的请求 |
| -40~-11 | 敌对 | 暗中阻挠，可能在考核中投反对票 |
| -40以下 | 死敌 | 主动攻击：弹劾、告状、造谣 |

### 1.3 好感度变化触发

| 行为 | 变化量 | 方向 | 备注 |
|------|--------|------|------|
| 考核中投支持票 | +15~+25 | 单向 | 可能产生人情债 |
| 做出对方认可的政绩 | +3~+8 | 单向 | 需政治理念相符 |
| 社交聚会同席 | +3~+5 | 双向 | |
| 送礼/帮忙办事 | +5~+15 | 单向 | |
| 偿还人情债 | +5~+10 | 单向 | 额外+社交信誉恢复 |
| 拒绝对方请求 | -5~-15 | 单向 | |
| 拒绝偿还人情债 | -20~-30 | 单向 | 额外：社交信誉大幅下降 |
| 弹劾/举报对方 | -30~-50 | 单向 | |
| 造谣中伤对方 | -20~-40 | 单向 | 若被发现，自身清名大降 |
| 公开颂扬对方 | +8~+15 | 单向 | 需好感度>20才自然 |
| 政治理念严重冲突决策 | -5~-10 | 单向 | |
| 参加对方政敌聚会 | -3~-8 | 单向 | |

## 2. 人情债系统

### 2.1 核心概念
- 人情债是官场社交的**核心货币**，不同于好感度
- 好感度是"喜不喜欢你"，人情债是"欠不欠你"
- 两者独立运作但互相影响

### 2.2 人情债的产生
重大帮助行为会产生人情债：

| 产生场景 | 债务等级 | 说明 |
|---------|---------|------|
| 考核中力挺（关键票，扭转结局） | 大恩 | 对方逃脱免职/罢黜 |
| 保人（对方被弹劾时出面担保） | 大恩 | 风险很高的行为 |
| 灾年借粮/借款救急 | 中恩 | 实质性经济帮助 |
| 举荐提拔 | 中恩 | 推动对方升迁 |
| 通风报信（提前告知危险） | 小恩 | 信息层面的帮助 |
| 社交场合公开维护 | 小恩 | 面子层面的帮助 |

### 2.3 人情债的偿还
- 债权人（帮助你的人）会在某个时间点**主动提出请求**
- 请求内容与债务等级对应：
  - 大恩 → 大请求（关照其亲属生意、考核中投支持票、帮忙压下弹劾）
  - 中恩 → 中请求（在某事上表态支持、借调人手或资金）
  - 小恩 → 小请求（参加其聚会、写推荐信）
- 玩家可以选择偿还或拒绝

### 2.4 偿还与拒绝的后果

| 行为 | 后果 |
|------|------|
| 偿还人情债 | 债务清除，好感度+5~+10，维护社交信誉 |
| 超额偿还 | 债务清除，好感度+10~+20，对方可能产生新的人情债给你 |
| 拒绝偿还 | 好感度-20~-30，社交信誉大幅下降，传闻扩散（其他Agent也知道你不还人情） |
| 拖延偿还 | 好感度每季度-3，对方可能升级请求或公开施压 |

### 2.5 社交信誉
- 隐藏值，范围0-100，初始60
- 影响其他Agent是否愿意帮你（信誉低→别人不敢帮你，怕你不还）
- 偿还人情债+5，拒绝-15，被传闻扩散后额外-10
- 信誉<30时，基本无人愿意为你冒风险

## 3. 中央政策表态

### 3.1 机制说明
- 中央（君主+内阁）每季度发起一项全国性政策议题
- MVP简化为**每年冬季1次**
- 各级官员需要表态：支持 / 反对 / 沉默

### 3.2 政策议题类型

| 类型 | 示例 | 涉及利益 |
|------|------|---------|
| 税赋调整 | 全国加税5% / 减税优惠 | 直接影响县域税收指标 |
| 军事行动 | 对北方动武 / 增兵边防 | 触发征兵征粮事件 |
| 基建工程 | 兴修全国运河 / 修缮长城 | 可能增加徭役摊派 |
| 科举改革 | 放宽录取 / 增加科目 | 影响本地文教和士子数量 |
| 反腐运动 | 全国清查贪腐 | 高风险高回报环境 |

### 3.3 表态的影响
- **支持**：与该政策推动者（君主或某派系）好感度+5~+10，但若政策不利于地方，可能损害民心或地方指标
- **反对**：与政策推动者好感度-5~-15，但与反对派好感度+5~+10；若反对一方败北，可能被秋后算账
- **沉默**：两边好感度各-2~-3，安全但没有收益；沉默过多次会被视为"骑墙派"，降低人缘
- 表态是重要的**站队信号**，影响派系判定

### 3.4 MVP实现
- 3年共3次政策表态事件
- 每次提供政策背景、各方立场、玩家选择界面
- 选择后触发各相关Agent的态度变化

## 4. 党派系统

### 4.1 MVP阶段：预设派系
- 开局时根据君主类型预设2-3个派系：

| 君主类型 | 派系设置 |
|---------|---------|
| 勤政型 | 保皇派（忠君、集权）vs 清流派（爱民、理想） |
| 怠政型 | 首辅派（掌实权）vs 次辅派（争权）vs 言官派（理想主义） |
| 中庸型 | 改革派（务实、分权）vs 守旧派（集权、现实） |

### 4.2 派系数据结构
```json
{
  "faction_id": "faction_001",
  "name": "保皇派",
  "leader": "agent_内阁首辅",
  "ideology": {
    "social_vs_people": 0.3,   // 偏社稷
    "centralize_vs_distribute": 0.2, // 偏集权
    "realistic_vs_ideal": 0.4  // 偏现实
  },
  "emperor_affinity": 75,
  "members": ["agent_001", "agent_003", ...],
  "rival_factions": ["faction_002"]
}
```

### 4.3 派系归属判定
Agent加入派系的条件：
- 政治理念与派系意识形态匹配（三维度均在同一区间，至少一维>0.7或<0.3）
- 且与派系领袖好感度>30
- 或被派系领袖主动招揽（好感度>50时自动尝试）

### 4.4 派系对游戏的影响
- **考核**：考核委员中的派系成员会倾向本派系官员
- **信息**：同派系成员更容易分享情报
- **互助**：同派系成员好感度天然+15
- **风险**：派系失势时所有成员受牵连（好感度连带下降，考核压力增大）

### 4.5 后续版本：动态派系形成
- 条件：不属于任何派系的内阁成员 + 朝中理念匹配且无派系的官员比例>某阈值
- 触发：自动形成以该内阁成员为首的新派系

## 5. 社交行动

### 5.1 行动清单

| 行动 | AP消耗 | 季节限制 | 效果 |
|------|--------|---------|------|
| 参加聚会 | 1 | 冬季为主 | 同席者好感+3~+5，获取1-2条传闻 |
| 写信问候 | 0（免费，每季度限1封） | 无 | 好感+2，维护关系 |
| 登门拜访 | 1 | 无 | 好感+5~+8，可能获取重要信息 |
| 送礼 | 0（消耗现金） | 无 | 好感+5~+15，清名-2（可能） |
| 举荐他人 | 1 | 考核季 | 被荐者好感+20，产生中恩人情债 |
| 弹劾/举报 | 2 | 无 | 关系破裂，若证据确凿可扳倒对方 |
| 造谣中伤 | 1 | 无 | 隐蔽攻击，降低对方名誉，但有被发现风险 |
| 公开颂扬 | 0 | 无 | 需好感度>20，好感+8~+15 |
| 中央政策表态 | 0（冬季强制） | 冬季 | 站队信号，详见第3节 |
| 请教师爷 | 0 | 无 | 每季度免费1次，获取分析建议 |

### 5.2 聚会机制
- 冬季社交季有2-3个聚会邀请，玩家选择参加哪个
- 每个聚会有一个东道主和3-5个参与者
- 聚会标签：同乡会 / 文人雅集 / 官场酒宴 / 派系私聚
- **聚会潜规则**：
  - 参加某人的聚会 = 一定程度上的站队信号
  - 东道主的政敌会知道你参加了，好感度-3~-5
  - 聚会中可触发对话事件（酒后交心、打探消息等）
  - 派系私聚信号更强烈：参加等同于半公开的派系站队

### 5.3 送礼系统
- 礼品分级：薄礼(10两,+5好感) / 厚礼(30两,+10好感) / 重礼(80两,+15好感)
- 有被拒绝的风险：清廉型Agent可能拒收并降低对你的评价
- 有被告发的风险：若送礼对象被弹劾，你可能被牵连
- 清名损耗：送出重礼清名-3；若被人传开额外-5

## 6. 情报与传闻系统

### 6.1 信息来源
| 来源 | 获取方式 | 可信度 |
|------|---------|--------|
| 师爷分析 | 免费，每季度1次 | 高（基于已知数据，但有师爷偏见） |
| 聚会传闻 | 参加聚会时获得 | 中（可能失真或有偏） |
| 下属汇报 | 衙役/村民代表主动上报 | 中低（可能报喜不报忧） |
| 密信 | 关系好的官员私下告知 | 高（需好感度70+） |
| 商户消息 | 拜访集市时获得 | 中（商人有利益立场） |
| 派系内部 | 同派系成员分享 | 较高（但可能有派系偏见） |

### 6.2 传闻内容类型
- **官场动向**："听说吏部张侍郎最近对南方几个县很不满"
- **人事变动**："据说知府大人明年可能调走"
- **他人评价**："邻县李县令私下说你施政过于激进"
- **政策风向**："朝中可能要推行新的税制改革"
- **地方情报**："王地主最近在偷偷买入东村的土地"
- **派系动向**："首辅派最近在拉拢兵部侍郎"

### 6.3 信息使用
- 传闻不直接显示真假，玩家需要交叉验证
- 同一信息被2个以上独立来源确认 → 可信度升至"高"
- 基于情报提前做出的决策可能有更好的结果

## 7. 关系网可视化（UI建议）

### 7.1 核心信息面板
- 列表展示已知NPC：名字、职位、关系状态（表情/颜色，非精确数字）、关系标签、人情债状态
- 精确好感度对玩家隐藏，只显示大致区间

### 7.2 关系图
- 简单的节点关系图：已知的NPC之间关系（同乡、师生、政敌、同派系）
- 随着玩家获取更多情报，逐步揭露更多连线和派系归属
- 人情债用特殊标记显示（如红色箭头=你欠他，绿色箭头=他欠你）
