# 07 — 趣味性与体验设计

---

## 1. 设计目标
AI Agent博弈是核心差异化卖点，但仅有博弈不够——需要丰富的"游戏感"层。
趣味性来源应覆盖：**策略深度、叙事沉浸、情感共鸣、惊喜感、成就感**。

---

## 2. 对话博弈（核心趣味模块）

### 2.1 审案系统——"明镜高悬"

**玩法**：地主与农民、商户之间的纠纷，玩家扮演审判官，通过对话查明真相。

**流程**：
```
案件描述（文字+简要背景）
  ↓
原告陈述（AI Agent）→ 玩家可追问
  ↓
被告辩护（AI Agent）→ 玩家可追问
  ↓
（可选）传唤证人 / 查验证据（消耗行动）
  ↓
玩家做出判决（自由文本或选项）
  ↓
系统评估判决合理性 → 后果结算
```

**趣味要素**：
- 双方都会说谎或隐瞒，需要通过追问找破绽
- 地主可能暗示"背后有人"施压
- 部分案件有隐藏的第三方因素（如真正的幕后黑手是邻县官员的亲属）
- 判决结果影响民心、地主关系、清名/威名

**MVP实现：3-5个案件模板**
| 案件类型 | 核心冲突 | 隐藏信息 |
|---------|---------|---------|
| 田产争夺 | 地主称祖传，农民称已耕三代 | 地契可能是伪造的 |
| 借贷纠纷 | 高利贷逼还vs借方确实困难 | 借方可能藏了钱 |
| 水源之争 | 上游村截水灌溉，下游村旱情 | 上游地主授意 |
| 伤人案 | 地主家仆打伤农民 | 可能涉及更大的欺压 |
| 商户欺诈 | 以次充好vs恶意索赔 | 第三方商业竞争阴谋 |

### 2.2 谈判系统——"虚与委蛇"

**场景举例**：
- 与知府谈减税额度
- 说服地主出资赈灾
- 与军事长官协调剿匪时间
- 接待京官，斡旋利益

**机制**：
- 对话有限轮数（5-8轮）
- 对方Agent有底线（如"最多减税5%"），但底线不透露
- 玩家可使用不同策略：晓之以理、动之以情、威逼、利诱
- 不同策略对不同性格的Agent效果不同
- 谈判破裂有后果（如知府不满、地主怀恨）

### 2.3 述职答辩——"金殿对策"

- 年终考核的面谈环节
- 委员问题有针对性（如"你任内税收下降了10%，作何解释？"）
- 玩家需要在有限时间内给出有说服力的回答
- 好的回答能扭转局面（低分翻盘），差的回答能毁掉好局（高分暴跌）

## 3. 两难困境精选（MVP 8-10个）

### 3.1 核心设计准则
- 无明显正确答案
- 每个选项牺牲不同的东西
- 事后会有后果提醒玩家"另一条路会怎样"

### 3.2 困境示例

**困境1：灾年抉择**
> 大旱，粮仓只够救一半人。县城有士绅豪商，是经济中心；乡村是耕种主力。
- A: 优先救县城 → 商业维持，但乡村民心崩溃，来年劳力不足
- B: 优先救乡村 → 来年耕种保障，但商业凋敝，士绅怨恨
- C: 平均分配 → 两边都不满意，谁都没救够

**困境2：恩师之托**
> 提携你的恩师来信，让你关照他侄子在本县的商铺（减免税收）。
- A: 照办 → 恩师好感大增，但清名-5，其他商户不满
- B: 婉拒 → 恩师好感-15，但维护公正形象
- C: 折中（给小恩小惠但不减税）→ 两边都不完全满意

**困境3：告密者**
> 你发现邻县县令贪腐的确凿证据，他和知府关系很好。
- A: 上报 → 清名+大，但得罪知府圈子，可能被报复
- B: 沉默 → 安全，但百姓继续受苦，你也知情不报
- C: 私下警告邻县县令 → 可能他收手（小概率），可能他先下手为强（大概率）

**困境4：站队时刻**
> 朝中推行新税法，知府坚决支持（因为能完成中央考核指标），但按察使认为会加重百姓负担。双方都在拉拢你表态。
- A: 支持知府 → 上级好感+，但执行后民心可能下降
- B: 支持按察使 → 清名+，但知府好感大降，考核可能受影响
- C: 含糊其辞 → 两边好感都略降，但不得罪任何一方

**困境5：地主的慷慨**
> 县里最大的地主主动提出捐资修桥（100两），条件是你"方便时关照一下他的田产纠纷案"。
- A: 接受捐资，暗中关照 → 桥修好(民心+，商业+)，但埋下隐患
- B: 接受捐资，但案件公正审理 → 地主感觉被耍，可能翻脸
- C: 拒绝 → 清名+，但桥没人修，百姓也不理解

## 4. 幕僚/师爷系统

### 4.1 定位
- 玩家的AI顾问团，降低上手门槛，同时增加策略层次
- 师爷本身也是Agent，有自己的性格和目标

### 4.2 MVP师爷（2选1开局）

| 类型 | 专长 | 性格 | 建议风格 |
|------|------|------|---------|
| 刑名师爷 | 审案、司法 | 刚正但死板 | 偏向严格执法，不考虑人情 |
| 钱粮师爷 | 经济、税收 | 圆滑务实 | 偏向利益最大化，有时建议灰色操作 |

### 4.3 交互方式
- 免费行动：每季度可以"问师爷"1次，获取对当前局面的分析
- 师爷会基于自己的专长和性格给建议，**不一定是最优解**
- 随着相处，师爷可能主动提醒你注意某些事（如"大人，那个地主最近动作频频"）
- 若长期违背师爷建议，师爷忠诚度下降，可能辞职

## 5. 因果报应系统——"三年之约"

### 5.1 核心理念
玩家在三年中的每一个决策都不会白费，最终的考核和结局会回溯所有关键决策。

### 5.2 实现方式
- 维护一个"决策日志"，记录每个关键决策
- 结局时生成"三年施政录"，逐一回顾关键决策及其长期影响
- 某些短期收益的决策，在第三年暴露出问题（如过度开垦导致水土流失）
- 某些短期牺牲的决策，在考核时成为加分项（如修水利的长期效果显现）

### 5.3 结局生成
```
基础结局（升迁/留任/免职/罢黜）
    +
声望侧写（清官/能吏/滑吏/铁腕/庸官等标签组合）
    +
三年施政录（关键决策回顾 + 连锁影响展示）
    +
历史人物类比（"你的施政风格最像___"）
    +
AI生成的个性化评价文本
```

## 6. 历史彩蛋与文化内容

### 6.1 决策后的历史闪回
- 当玩家做出某个典型决策后，弹出简短历史知识卡片
- 例：强行赈灾后 → "海瑞在淳安县任上也曾如此…"（30字简介）
- 不强制阅读，作为可展开的补充内容

### 6.2 历史人物类比结局
| 类型 | 对应原型 | 触发条件 |
|------|---------|---------|
| 清廉楷模 | 海瑞 | 清名>80，能名中等，人缘低 |
| 改革能臣 | 张居正 | 能名>80，威名>60，但可能清名不高 |
| 老好人 | 申时行 | 人缘>80，各项中庸 |
| 权谋高手 | 严嵩 | 人缘>70，权力目标最大化，灰色操作多 |
| 隐士之风 | 王阳明早期 | 主动隐退，文教贡献大 |

### 6.3 科举知识小游戏
- 赴任途中或社交场合，偶尔触发诗词/策论挑战
- 成功完成可获得+文教或+社交好感
- 不影响核心流程，纯粹增加趣味和文化沉浸

## 7. 随机奇遇事件

为避免游戏过于严肃，加入少量趣味性随机事件：

| 事件 | 内容 | 效果 |
|------|------|------|
| 游方郎中 | 一位神秘郎中到访，可治疗县内瘟疫 | 瘟疫事件快速解决 |
| 藏书发现 | 县学扩建时发现一批古籍 | 文教+5，可能引发"文人雅集" |
| 民间发明 | 一位工匠发明了新农具 | 可选择推广（亩产+5%）或忽视 |
| 奇案异闻 | 离奇的案件（非常规纠纷） | 纯叙事趣味，选择影响民间传说 |
| 外邦商人 | 外地商队路过 | 可选择招商(商业+)或驱赶(治安+) |

## 8. UI/UX体验建议

### 8.1 信息呈现
- **主界面**：县域简略地图 + 四季时间轴 + 行动力指示 + 事件通知
- **状态面板**：各指标用图标+描述性文字，非数字（如民心用表情图标）
- **对话界面**：类似聊天界面，NPC有头像和情绪指示
- **决策界面**：两难选择用分屏对比展示各选项的预期后果

### 8.2 节奏控制
- 每季度操作时间控制在5-8分钟
- 对话事件控制在3-5分钟
- 一局游戏（3年12季度）约60-90分钟
- 允许存档，每季度结束时自动保存
