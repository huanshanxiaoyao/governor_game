# 05 — 考核与升迁系统

> v0.2 更新：对齐构思文档的考核委员会组成、人事流动规则

---

## 1. 考核节奏

### 1.1 年度小考
- 时间：每年冬季
- 形式：知府对下辖各县进行书面评审
- 结果：优/良/中/差，影响知府对你的印象值和后续大考

### 1.2 三年大考
- 时间：第3年冬季（MVP只有一次大考）
- 形式：考核委员会集体评审 + 面谈（对话事件）
- 结果：升迁/留任/免职/罢黜

---

## 2. 考核委员会

### 2.1 县州级官员考核委员会（5人）
| 席位 | 人选 | 倾向说明 |
|------|------|---------|
| 本省省级官员A | 巡抚或布政使 | 倾向看政绩和税收 |
| 本省省级官员B | 按察使 | 倾向看司法和治安 |
| 本府知府 | 玩家直属上级 | 综合考虑，好感度影响大 |
| 吏部侍郎 | 中央代表 | 看大局，受上层政治和派系影响 |
| 外省省级官员 | 随机选1名 | 较客观，但可能有隐藏关系网 |

### 2.2 府/省级官员考核委员会（后续版本，5-6人）
| 席位 | 人选 |
|------|------|
| 吏部尚书 | ×1 |
| 吏部侍郎 | ×3 |
| 内阁成员 | 随机×1 |
| 朝中一品虚职 | 随机×1（代表老臣或某势力话事人） |
| 君主 | 对升迁有一票否决权 |

### 2.3 投票机制
- 每位委员独立给出评价：升迁 / 留任 / 免职 / 罢黜
- 投票权重：吏部侍郎×1.5，其余各×1.0
- 加权汇总后取结果
- 县级考核：君主通常不干预（低概率触发否决）
- 府/省级考核：君主对升迁有一票否决权

## 3. 考核评分模型

### 3.1 客观分（权重50%）
基于三年施政数据：

| 指标 | 权重 | 评分标准 |
|------|------|---------|
| 税收完成率 | 25% | 三年累计上缴 ÷ 指标总额 |
| 民心变化 | 20% | 三年末 vs 初始值的变化 |
| 治安变化 | 15% | 同上 |
| 人口变化 | 15% | 同上 |
| 商业发展 | 10% | 同上 |
| 文教发展 | 10% | 同上 |
| 重大事故 | 5% | 任内是否发生暴动、大规模流民等 |

### 3.2 主观分（权重50%）
由考核委员AI Agent基于以下因素打分：

- **与委员的好感度**：好感度高的委员倾向打高分
- **委员个人偏好**：
  - 重视清名的委员：看你是否廉洁、拒绝贿赂
  - 重视能名的委员：看政绩数据
  - 重视人缘的委员：看你的社交网络和口碑
  - 重视忠君的委员：看你执行中央政策的力度
- **他人举荐/弹劾**：收到的关于你的正面或负面信息
- **面谈表现**：对话事件中你的应对（由AI评判）

### 3.3 评分合成
```
最终评分 = 客观分 × 0.5 + 主观分平均 × 0.5

90+  → 升迁（优先候选）
70-89 → 留任（表现良好）
50-69 → 留任（勉强合格）
30-49 → 免职（待命，等待重新任命）
<30  → 罢黜（停薪停职）
```

## 4. 年终面谈设计（对话事件）

### 4.1 流程
1. 开场：委员宣布考核开始，简述你的任职情况
2. 自述环节：玩家概述三年政绩（自由文本输入）
3. 质询环节：2-3位委员各提一个问题（AI生成）
   - 问题类型：质疑某项决策的原因、追问某个指标不佳的理由、考察施政理念
4. 结束：委员做最终评定

### 4.2 面谈技巧对结果的影响
- 回答切中委员关心的点 → 该委员主观分+10~+15
- 回答诚恳但无亮点 → 无变化
- 回答引起委员反感（如过于狂妄、推卸责任）→ 该委员主观分-10~-20
- 说出委员不知道的好信息（如某项隐性政绩）→ 可能翻盘

## 5. 升迁路径（MVP后扩展）

### 5.1 县令之后的路径
```
七品县令 → 六品知州（平调到更大的县级单位）
         → 五品省司官（布政司/按察司参议）
         → 四品知府
         → 二品巡抚
         → 一品总督/入阁
```

### 5.2 非升迁出路
- **免职待命**：保留品级薪俸，无职位，等待0.5-2年后重新任命
- **罢黜**：停薪停职，**游戏失败（Game Over）**
- **主动隐退**：保留名声，等待政治环境变化后复出（后续版本）

## 6. 人事流动规则

### 6.1 新人加入
- 每年新一批科考上岸人员加入
- 默认从八品做起，担任各地县令的助手
- 新人籍贯随机，与本地县令形成同僚关系

### 6.2 退休规则
- 60岁默认退休，除非省级/内阁/六部尚书点名继续任用
- 一二品官员退休年龄放宽到66岁
- 非内阁的一二品官员过60岁，默认转任虚职
- 内阁成员无年龄强制退休（但君主可随时调整）

### 6.3 对玩家的影响
- 新人可能成为你的下属（后续版本中知府玩法）
- 退休事件可能腾出职位，创造升迁机会
- 恩师/靠山退休意味着失去保护伞

---

## 7. MVP中的考核实现

### 6.1 简化方案
- 第1年、第2年末：只做知府的年度小考（知府给一段评语 + 优良中差评级）
- 第3年末：完整大考流程（委员会+面谈+投票）
- 大考结果作为MVP的终局评价核心依据之一

### 6.2 考核结果与结局联动
- 升迁 → 结局类型"前途光明"
- 留任（高分）→ 结局类型"稳健之官"
- 留任（低分）→ 结局类型"碌碌无为"
- 免职 → 结局类型"壮志未酬"
- 罢黜 → 结局类型"官场沉浮"
- 各结局再叠加声望维度，生成最终的历史评价文本
