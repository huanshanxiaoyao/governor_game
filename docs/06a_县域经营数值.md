# 06a — 县域经营数值

> 子文档，隶属于 [06 — 数值体系设计](06_数值体系设计.md)
>
> 本文档涵盖县域类型、指标定义、核心公式与经济平衡参照。

## 1. 县域指标定义

### 1.1 状态指标（0-100）

| 指标 | 含义 | 自然趋势 | 关键阈值 |
|------|------|---------|---------|
| 民心 | 百姓对官府的认可度 | 每季度-1（自然衰减） | <25暴动风险，>75减少治安事件 |
| 治安 | 地区安全程度 | 每季度-1 | <20匪患频发，>70吸引商户 |
| 商业 | 商业活跃程度 | 无自然变化 | >70触发繁荣事件，<15集市凋敝 |
| 文教 | 教育文化水平 | 无自然变化 | >60概率出才子，>80成为文教名县 |

### 1.2 经济指标

| 指标 | 单位 | 说明 |
|------|------|------|
| 人口 | 人 | 各村合计（在册民户） |
| 耕地面积 | 亩 | 含自耕地和地主占地 |
| 地主占地比 | % | 地主/宗族/绅衿占有的土地比例，影响徭役税基 |
| 县库现金 | 两 | 可用资金，负数代表欠债 |
| 年度农业产出 | 两 | 秋季计算 |
| 年度税收 | 两 | 农业税+徭役折银+商业税 |
| 年度支出 | 两 | 行政+投资+上缴+突发 |

### 1.3 县域类型

游戏开局随机选取一种县域类型，基于类型设定初始参数（详见§6）。

| 类型 | 分布地域 | 核心矛盾 |
|------|---------|---------|
| 财赋核心型 | 江南太湖平原（苏、松、常、杭、嘉、湖） | 高额赋税定额与地主占地集中的冲突，徭役负担集中在少数平民 |
| 宗族治理型 | 皖南徽州、赣东南抚州等山区 | 宗族势力与官府权力的博弈，宗族主导基层秩序 |
| 沿海治理型 | 闽、浙、粤沿海 | 海防安全与民生发展的平衡，财政紧张 |
| 黄淮灾荒型 | 黄河、淮河中下游沿线 | 灾荒频发与固定赋税定额的冲突，流民问题突出 |

**类型对gameplay的影响**：
- **财赋核心型**：钱多事也多，上缴压力大，地主占地比高导致徭役税基窄、平民负担重
- **宗族治理型**：社会秩序稳定，税收征收效率高，但宗族势力强，改革阻力大
- **沿海治理型**：人少地少钱少，海防开支大，收支勉强持平，容错空间极小
- **黄淮灾荒型**：耕地多但灾害频繁，水患风险极高，需要持续投入水利和赈灾

### 1.4 物价指数 (Price Index)

每种县域类型有一个固定的物价指数（`price_index`），反映当地物价水平对施政成本的影响。

| 县域类型 | price_index | 说明 |
|---------|-------------|------|
| 财赋核心型 | 1.4 | 江南物价高，一切投资和开支都贵 |
| 宗族治理型 | 1.1 | 接近基准物价 |
| 沿海治理型 | 0.9 | 偏远地区物价稍低 |
| 黄淮灾荒型 | 0.8 | 中部物价较低 |

**物价指数影响范围**：
- **投资项目成本**：所有8种投资类型的花费按 `round(基础cost × price_index)` 缩放
- **增设衙役的增量行政开支**：每级衙役增加的行政开支 `+round(40 × price_index)` 两
- **医疗开支**：年度医疗费用按 `round(医疗基础费用 × price_index)` 缩放

**物价指数不影响**：
- **基础行政开支（admin_cost）**：各县域类型的初始行政开支已按地区差异化设定（200/150/100/130两），隐含了物价水平
- **税收收入**：农业税、徭役折银、商业税均为产出或人口驱动，不受本地物价影响
- **农业产出**：亩产为实物产出折算，独立于物价

**设计意图**：物价指数为县域类型增加一层经济差异化——
- 财赋核心型（1.4）：投资成本高40%，加剧"钱多但花得更多"的困境
- 沿海治理型（0.9）：投资略便宜，部分补偿其极度紧张的财政
- 黄淮灾荒型（0.8）：投资便宜20%，为灾后重建降低门槛

**后续可扩展**（MVP不实现）：
- 动态物价波动：`price_index` 可按 ±0.05/年 随商业指数和灾害变化
- 影响徭役折银：物价越高，折银额应越高
- 影响商业税基：商户在高物价地区营收更高

## 2. 土地与农业

### 2.1 农业产出
```
村庄产出 = 耕地面积 × 基础亩产 × 农业适宜度 × (1 + 水利加成) × (1 - 灾害损失)

基础亩产 = 2两/亩
农业适宜度 = 0.3 ~ 1.0（全局环境参数，年初确定）
水利加成 = 0 ~ 0.3（取决于水利建设等级）
灾害损失 = 0 ~ 0.7（取决于灾害类型和减灾措施）
```

### 2.2 水灾概率
```
水灾触发概率 = 水患风险度 ×  (1 - 水利等级 × 0.2)

水利等级：
  0 = 无水利设施
  1 = 基础水利（投资100两建设）
  2 = 完善水利（追加200两升级）

例：水患风险度0.8，无水利 → 概率24%
    水患风险度0.8，完善水利 → 概率24% × (1-1.0) = 0%（上限封顶）
    实际公式封底：min(概率, 0.02)，即最低2%的不可消除风险
```

## 3. 人口

### 3.1 人口变化
```
季度人口变化 = 自然增长 + 流入 - 流出 - 灾害/征兵损失

自然增长 = 人口 × 0.005（季度）
流入 = 商业指数>50时，每季度+10~30人
流出 = 民心<30时，人口×0.02流失
       治安<20时，额外人口×0.03流失
```

### 3.2 征兵影响
```
征兵强度 = 边患风险度 × 基准征调比例(2%)
征调人口 = 总人口 × 征兵强度
影响：
  人口减少 = 征调人口
  民心 -= 征调人口 / 总人口 × 30
  耕地产出 -= 征调人口 / 总人口 × 0.5（劳动力减少）
```

## 4. 财政体系

### 4.1 农业税
```
农业税 = 总耕地产出 × 税率 × 征收效率

  总耕地产出 = 耕地面积 × 基础亩产 × 农业适宜度 × (1 + 水利加成) × (1 - 灾害损失)
  税率 = 中央基准(12%) ± 玩家调整(±3%)
  征收效率 = 0.7 ~ 1.0（受民心、衙役数量影响）

说明：地主和自耕农适用相同税率，对所有在册土地统一征收。
（地主隐匿土地属于独立事件机制，不在基础税收公式中体现。）
```

### 4.2 徭役折银
```
徭役折银 = 应役人口 × 人均徭役折银额

  应役人口 = 总人口 × (1 - 地主人口比例)
  地主人口比例 ≈ 地主占地比 × 0.12（地主人少地多，占总人口比例远低于占地比）
  人均徭役折银额 = 0.3两/人·年

说明：地主（绅衿）依制免除徭役。地主占地比越高，免役人口越多，
应役人口越少 → 徭役总收入越低，且剩余平民人均徭役负担越重。

示例：
  人口5000，地主占地比60% → 地主人口比例7.2% → 应役人口4640
  徭役折银 = 4640 × 0.3 = 1392两

  人口5000，地主占地比35% → 地主人口比例4.2% → 应役人口4790
  徭役折银 = 4790 × 0.3 = 1437两
```

### 4.3 商业税
```
商业税 = Σ(各集市: 商户数 × 基准税额 × 商业指数/50)
  基准税额 = 5两/商户（受经济气候修正）
```

### 4.4 财政汇总
```
总税收 = 农业税 + 徭役折银 + 商业税

上缴额 = 总税收 × 上缴比例
  上缴比例：因县域类型不同（见§6），可通过与知府谈判调整±5%

县域留存 = 总税收 - 上缴额
```

## 5. 指标季度变化

### 5.1 民心变化
```
民心变化 = 基础衰减(-1)
          + 赈灾行动加成(0~+8)
          + 减税效果(0~+5)
          + 文教贡献(文教指数/20, 即0~+5)
          - 重税惩罚(超额征税时 -3~-10)
          - 灾害未救助(-5~-15)
          - 征兵影响(-2~-8)
          + 随机事件影响
```

### 5.2 治安变化
```
治安变化 = 基础衰减(-1)
          + 衙役加成(衙役等级 × 2, 即0~+6)
          + 民心联动(民心>60时+1, <30时-2)
          - 流民影响(每100流民 -2)
          + 剿匪成功(+5~+10)
          - 征兵后治安空白(-2~-5)
          + 随机事件影响
```

## 6. 经济平衡参照表

### 6.1 县域类型初始参数

生成规则：以下基准值在每局游戏创建时，各数值增加 **±20%相对值**的随机波动。

> 地主占地比基准 = 历史史料数据 × 0.7（史料偏明朝后期土地兼并高峰，
> 游戏默认设定偏早期；后续可根据每局君主类型进一步差异化）。

> **实现说明**：代码中耕地面积使用 ×4 缩放约定（配合 `MAX_YIELD_PER_MU=200斤/亩`
> 和 `base_yield=0.5两/亩` 的产出模型）。本文档中的耕地数值为**设计值**，
> 代码中的实际存储值 = 设计值 × 4。经济产出公式已相应调整，最终结果一致。

### 6.2 收支推演：财赋核心型（风调雨顺，适宜度0.9）
```
农业税 = 15000 × 2 × 0.9 × 12% × 0.85 ≈ 2754两
徭役折银 = 8000 × (1 - 0.63×0.12) × 0.3 ≈ 2218两
商业税 = 25商户 × 5 × 55/50 ≈ 138两
总税收 ≈ 5110两

上缴 = 5110 × 75% ≈ 3833两
行政 ≈ 200两
县域留存 = 5110 - 3833 - 200 + 600(存量) ≈ 1677两
特点：收入高但上缴压力极大，留存空间取决于谈判能力
```

### 6.3 收支推演：宗族治理型（正常年景，适宜度0.7）
```
农业税 = 13000 × 2 × 0.7 × 12% × 0.9 ≈ 1966两
徭役折银 = 6000 × (1 - 0.55×0.12) × 0.3 ≈ 1681两
商业税 = 10商户 × 5 × 30/50 ≈ 30两
总税收 ≈ 3677两

上缴 = 3677 × 65% ≈ 2390两
行政 ≈ 150两
县域留存 = 3677 - 2390 - 150 + 400(存量) ≈ 1537两
特点：征收效率高（宗族配合），财政健康但商业薄弱
```

### 6.4 收支推演：沿海治理型（正常年景，适宜度0.6）
```
农业税 = 5000 × 2 × 0.6 × 12% × 0.8 ≈ 576两
徭役折银 = 3000 × (1 - 0.36×0.12) × 0.3 ≈ 861两
商业税 = 12商户 × 5 × 40/50 ≈ 48两
总税收 ≈ 1485两

上缴 = 1485 × 60% ≈ 891两
行政 ≈ 100两
县域留存 = 1485 - 891 - 100 + 150(存量) ≈ 644两
特点：收支紧平衡，一次海患或灾害即可能破产
```

### 6.5 收支推演：黄淮灾荒型（灾年，适宜度0.4）
```
农业税 = 12000 × 2 × 0.4 × 12% × 0.75 ≈ 864两
徭役折银 = 5000 × (1 - 0.47×0.12) × 0.3 ≈ 1416两
商业税 = 6商户 × 5 × 20/50 ≈ 12两
总税收 ≈ 2292两

上缴 = 2292 × 65% ≈ 1490两
行政 ≈ 130两
县域留存 = 2292 - 1490 - 130 + 250(存量) ≈ 922两
特点：灾年农业税骤降但上缴不减，需申请减免或动用县库赈灾
      若同时发生水灾，赈灾支出200~500两，财政立刻见底
```
