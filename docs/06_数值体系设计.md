# 06 — 数值体系设计

> v0.2 更新：新增全局环境系统、对齐三层属性、优化信息不透明设计

---

## 1. 设计原则
- **可感知**：玩家能通过游戏反馈直觉理解数值变化，而非依赖查看精确数字
- **有张力**：多个指标之间存在天然矛盾，不可能全部最优
- **信息迷雾**：大量数值半隐藏或隐藏，玩家需要通过观察和社交来推断
- **防通刷**：没有单一最优策略，不同局面需要不同应对

## 2. 全局环境参数

### 2.1 地理气候（每年微调）

| 参数 | 取值范围 | 每年变化 | 影响 |
|------|---------|---------|------|
| 农业适宜度 | 0.0-1.0 | ±0.1随机 | 乘算到基础亩产 |
| 水患风险度 | 0.0-1.0 | ±0.1随机 | 水灾触发概率 = 水患风险度 × 0.3 × (1 - 水利等级×0.5) |
| 边患风险度 | 0.0-1.0 | ±0.05随机 | 影响中央征兵事件的强度和频率 |

### 2.2 年初暗示（信息呈现方式）
不直接告知数值，而是以叙述形式暗示：
- 农业适宜度高 → "今春风调雨顺，老农皆言是个好年景"
- 水患风险高 → "入夏以来雨水偏多，堤坝需多加留意"
- 边患风险高 → "北方边报频传，朝中气氛紧张"
- 农业适宜度低 → "开春以来旱象初现，不少田地未能按时播种"

### 2.3 经济气候（MVP不动态变化）

| 等级 | 影响 |
|------|------|
| 繁荣 | 商业基础指数+15，商户数+20%，基准商业税额+2两 |
| 平稳 | 无加成 |
| 萧条 | 商业基础指数-10，商户流失风险每年5%，基准商业税额-1两 |

### 2.4 政治气候（MVP不动态变化）
- 由君主类型决定，影响全局行为模式：

| 类型 | 考核侧重 | Agent行为倾向 |
|------|---------|-------------|
| 勤政高压 | 税收完成率权重+10%，反腐指标 | 贪腐Agent更隐蔽，清官更受重用 |
| 怠政党争 | 派系站队权重+15%，政绩权重降低 | 社交和站队比施政更重要 |
| 中庸开明 | 各维度均衡 | 正常博弈环境 |

## 3. 县域核心指标

### 3.1 指标定义（0-100，除人口和资金外）

| 指标 | 含义 | 自然趋势 | 关键阈值 |
|------|------|---------|---------|
| 民心 | 百姓对官府的认可度 | 每季度-1（自然衰减） | <25暴动风险，>75减少治安事件 |
| 治安 | 地区安全程度 | 每季度-1 | <20匪患频发，>70吸引商户 |
| 商业 | 商业活跃程度 | 无自然变化 | >70触发繁荣事件，<15集市凋敝 |
| 文教 | 教育文化水平 | 无自然变化 | >60概率出才子，>80成为文教名县 |

### 3.2 经济指标

| 指标 | 单位 | 说明 |
|------|------|------|
| 人口 | 人 | 各村合计 |
| 耕地面积 | 亩 | 含自耕地和地主占地 |
| 县库现金 | 两 | 可用资金，不可为负 |
| 年度农业产出 | 两 | 秋季计算 |
| 年度税收 | 两 | 农业税+商业税 |
| 年度支出 | 两 | 行政+投资+上缴+突发 |

### 3.3 玩家能力值（后台追踪，1-10）
玩家不可见精确值，系统根据行为累积：

| 能力 | 初始值 | 提升方式 | 影响 |
|------|--------|---------|------|
| 施政力 | 3-5（随出身） | 成功执行政策+0.2 | 政策执行效率乘数 |
| 明察力 | 3-5 | 成功审案+0.3 | 审案准确度、识破谎言概率 |
| 辩才 | 3-5 | 对话表现好+0.2 | 对话博弈成功率加成 |
| 统御力 | 3-5 | 成功维持治安+0.2 | 衙役效率、地主配合度加成 |

### 3.4 声望维度（半隐藏，0-100）

| 维度 | 增加方式 | 减少方式 |
|------|---------|---------|
| 清名 | 拒绝贿赂、公正审案 | 送礼、收礼、偏袒 |
| 能名 | 政绩提升、危机处理 | 施政失误、指标下降 |
| 人缘 | 社交活跃、帮助同僚 | 弹劾他人、拒绝请求 |
| 威名 | 铁腕执法、平定匪患 | 妥协退让 |

**展示方式**：5档分级（极低/低/中/高/极高），但有±10的随机偏差。
玩家看到的不一定是真实等级，需要通过传闻来交叉验证自己的真实声望。

## 4. 核心公式

### 4.1 农业产出
```
村庄产出 = 耕地面积 × 基础亩产 × 农业适宜度 × (1 + 水利加成) × (1 - 灾害损失)

基础亩产 = 2两/亩
农业适宜度 = 0.3 ~ 1.0（全局环境参数，年初确定）
水利加成 = 0 ~ 0.3（取决于水利建设等级）
灾害损失 = 0 ~ 0.7（取决于灾害类型和减灾措施）
```

### 4.2 税收
```
农业税 = Σ(各村产出) × 税率 × 征收效率
  税率 = 中央基准(12%) ± 玩家调整(±3%)
  征收效率 = 0.7 ~ 1.0（受民心、统御力、衙役数量影响）

商业税 = Σ(各集市: 商户数 × 基准税额 × 商业指数/50)
  基准税额 = 5两/商户（受经济气候修正）

上缴比例 = 总税收 × 65%（可通过与知府谈判调整±5%）
县域留存 = 总税收 - 上缴额
```

### 4.3 水灾概率
```
水灾触发概率 = 水患风险度 × 0.3 × (1 - 水利等级 × 0.5)

水利等级：
  0 = 无水利设施
  1 = 基础水利（投资100两建设）
  2 = 完善水利（追加80两升级）
  
例：水患风险度0.8，无水利 → 概率24%
    水患风险度0.8，完善水利 → 概率24% × (1-1.0) = 0%（上限封顶）
    实际公式封底：min(概率, 0.02)，即最低2%的不可消除风险
```

### 4.4 征兵影响
```
征兵强度 = 边患风险度 × 基准征调比例(2%)
征调人口 = 总人口 × 征兵强度
影响：
  人口减少 = 征调人口
  民心 -= 征调人口 / 总人口 × 30
  耕地产出 -= 征调人口 / 总人口 × 0.5（劳动力减少）
```

### 4.5 指标季度变化
```
民心变化 = 基础衰减(-1)
          + 赈灾行动加成(0~+8)
          + 减税效果(0~+5)
          + 文教贡献(文教指数/20, 即0~+5)
          - 重税惩罚(超额征税时 -3~-10)
          - 灾害未救助(-5~-15)
          - 征兵影响(-2~-8)
          + 随机事件影响

治安变化 = 基础衰减(-1)
          + 衙役加成(衙役等级 × 2, 即0~+6)
          + 民心联动(民心>60时+1, <30时-2)
          - 流民影响(每100流民 -2)
          + 剿匪成功(+5~+10)
          - 征兵后治安空白(-2~-5)
          + 随机事件影响
```

### 4.6 人口变化
```
季度人口变化 = 自然增长 + 流入 - 流出 - 灾害/征兵损失

自然增长 = 人口 × 0.005（季度）
流入 = 商业指数>50时，每季度+10~30人
流出 = 民心<30时，人口×0.02流失
       治安<20时，额外人口×0.03流失
```

## 6. 经济平衡参照表（MVP中等难度县）

### 6.1 典型初始值
```
人口：5000人 | 耕地：10000亩（地主占35%）| 县库：400两
民心：50 | 治安：55 | 商业：35 | 文教：25
农业适宜度：0.7 | 水患风险：0.4 | 边患风险：0.2
```

### 6.2 典型年度收支（风调雨顺，适宜度0.9）
```
农业税 = 10000亩 × 2两 × 0.9 × 12% × 0.85 ≈ 1836两
商业税 = 15商户 × 5两 × 35/50 ≈ 53两
总税收 ≈ 1889两

上缴 = 1889 × 65% ≈ 1228两
行政 ≈ 120两
可投资 = 1889 - 1228 - 120 + 400(存量) ≈ 941两
扣除应急储备200两 → 实际可投资约700两
```

### 6.3 灾年收支（旱灾，适宜度0.5）
```
农业税 ≈ 10000 × 2 × 0.5 × 12% × 0.8 ≈ 960两
上缴仍要求 ≈ 1228两
缺口 = 1228 + 120 - 960 = 388两（必须动用县库或申请减免）
```

### 6.4 征兵年收支（边患风险0.8）
```
征调人口 = 5000 × 0.8 × 2% = 80人
人口降至4920 → 耕地产出减少约1.6%
民心额外 -5，治安额外 -3
```

## 7. 信息不透明设计

### 7.1 对玩家显示的信息

| 信息类型 | 展示方式 | 精确度 |
|---------|---------|--------|
| 县库现金 | 精确数值 | 100% |
| 人口 | 精确数值 | 100% |
| 耕地面积 | 精确数值 | 100% |
| 民心/治安/商业/文教 | 5档描述（极差/低迷/尚可/良好/繁荣） | ±1档偏差可能 |
| 趋势 | ↑↓→ 箭头 | 方向准确，幅度不明 |
| 全局环境 | 叙述性暗示文字 | 模糊 |
| 自身声望 | 5档分级 | ±1档偏差 |
| NPC好感度 | 不显示 | — |
| NPC性格/理念 | 不显示 | — |
| 考核评分权重 | 不显示 | — |
| 链式事件埋点 | 不显示 | — |

### 7.2 获取更精确信息的途径
- **师爷分析**：可以得到更精确的县域指标估计（偏差缩小到±5%）
- **社交传闻**：了解NPC对你的大致态度和政治倾向
- **派系情报**：同派系成员可能透露考核委员的偏好
- **多次交互**：与同一NPC多次互动后，系统逐步揭露其部分性格标签

### 7.3 设计意图
信息不透明是核心游戏体验的一部分：
- 迫使玩家投资社交来获取信息 → 社交行动有了策略价值
- 决策带有不确定性 → 增加博弈深度和紧张感
- 鼓励多周目游玩 → 第二局能更好地解读信号
